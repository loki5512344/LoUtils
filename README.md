## 🍃 FastLeafDecay

Листва исчезает за 2 секунды после рубки дерева.

### ✅ Исправлено в v2.3.1
- Stripped logs теперь работают корректно
- Листья, поставленные игроком (persistent), не удаляются
- Листья не удаляются через воздух (только связанные)
- Оптимизирован BFS алгоритм (в 4 раза быстрее)

### Команды
```bash
/lfastleaf enable          # Включить
/lfastleaf disable         # Выключить
/lfastleaf status          # Статус
/lfastleaf reload          # Перезагрузить
/lfastleaf set delay 40    # Задержка (тики)
/lfastleaf set radius 5    # Радиус поиска
/lfastleaf set animation 2 # Анимация
/lfastleaf set smart true  # Умный алгоритм
```

### Конфиг: `conf/fastleafdecay.yml`
```yaml
enabled: true
decay-delay: 40        # Задержка перед гниением (тики)
search-radius: 5       # Радиус поиска листвы
animation-delay: 2     # Задержка между удалением листьев
smart-decay: true      # Умный алгоритм
```

### Особенности
- Удаляет только естественные листья (не поставленные игроком)
- Проверяет связь листьев с брёвнами (не удаляет через воздух)
- Работает со всеми типами деревьев, включая stripped logs
- Оптимизированный алгоритм поиска

## 🔥 CauldronCrafting

Котёл как станция переработки.

### Способы использования
- **ПКМ по котлу** с предметом в руке
- **Бросить предмет** в котёл (Q)

### Рецепты

#### 1. Очистка бетона
`Цементный порошок` → `Бетон` (все 16 цветов)

#### 2. Стирка (удаление краски)
- Кожаная броня → без краски
- Цветной флаг → белый флаг
- Цветная кровать → белая кровать

**Конфиг:** `conf/cauldron.yml`

## 💡 LightBlock

Невидимый источник света (уровень 0-15).

### Крафт
```
[Светокамень] [Светокамень] [Светокамень]
[Светокамень] [  Свечка  ] [Светокамень]
[Светокамень] [Светокамень] [Светокамень]
= Блок света (уровень 15)
```

### Использование
- **ПКМ** — поставить блок
- **Shift + ПКМ по блоку** — изменить уровень (0-15)
- **ПКМ с блоком в руке** — удалить блок
- **Shift + ПКМ в воздух** — изменить уровень в руке

### Видимость
- **Креатив/Спектатор** — видят нативно (как барьеры)
- **Выживание** — видят частицы GLOW (радиус 16 блоков)

**Конфиг:** `conf/light-block.yml`

## 🖼️ InvisibleFrames

Невидимые рамки (остаётся только предмет).

### Крафты

#### 1. Верстак
```
[Рамка] [Рамка] [Рамка]
[Рамка] [Зелье ] [Рамка]
[Рамка] [Рамка] [Рамка]

Зелье невидимости + 8 рамок = 8 невидимых рамок
```
Работает с любым зельем невидимости (обычное/splash/lingering).

#### 2. Бросок зелья
Бросьте splash/lingering зелье невидимости на рамку.

**Конфиг:** `conf/invisible-frames.yml`

## 🎭 Player Poses

Система поз (аналог GSit).

### Команды
- `/sit` — сесть на месте
- `/pose stop` — встать

### Автосидение
- ПКМ по ступенькам/плитам/коврам
- ПКМ по игроку (сесть на голову)

### Конфиг: `conf/poses.yml`
```yaml
sit:
  enabled: true
  right-click-enabled: true
  sit-height: -1.5
general:
  cancel-on-damage: true
  cancel-on-move: true
player-sit-enabled: true
```

**Права:** `loutils.sit`, `loutils.pose`

## 🗺️ Map Locking

Защита карт от копирования.

### Команды
- `/map lock` — заблокировать (только владелец может разблокировать)
- `/map unlock` — разблокировать

### Механика
- UUID владельца в NBT
- Имя владельца в описании
- Защита от копирования в картографическом столе

**Конфиг:** `conf/map-locking.yml`

---

## 🖼️ Frame Locking

Защита рамок мёдом.

### Механика
- **ПКМ с мёдом** — заблокировать
- **ПКМ с кистью** — разблокировать

### Защита
- Нельзя повернуть предмет
- Нельзя достать предмет
- Нельзя сломать рамку

**Конфиг:** `conf/frame-locking.yml`

## 🌱 Enhanced Bone Meal

Костная мука работает на тростнике и кактусах (порт с Bedrock).

### Механика
- ПКМ по тростнику → +1-3 блока
- ПКМ по кактусу → +1-3 блока
- Максимальная высота: 3 блока
- Шанс успеха: 75%

**Конфиг:** `conf/enhanced-bone-meal.yml`

## 🔨 Anvil Repair

Починка наковален железными блоками.

### Механика
- `Damaged Anvil` + железный блок → `Chipped Anvil`
- `Chipped Anvil` + железный блок → `Anvil`

**Конфиг:** `conf/anvil-repair.yml`

---

## ✂️ Name Tag Removal

Снятие бирок с мобов ножницами.

### Механика
- ПКМ ножницами по мобу с биркой
- Бирка дропается с именем моба
- Не работает на боссах (настраивается)

**Конфиг:** `conf/name-tag-removal.yml`

## 🎮 Команды и права

### Основные команды

| Команда | Описание | Право |
|---------|----------|-------|
| `/lw <add\|remove\|list>` | Whitelist | `loutils.whitelist` |
| `/lar <start\|stop\|status>` | Авто-рестарт | `loutils.autorestart` |
| `/lspawnmob <mob> <amount>` | Спавн мобов | `loutils.spawnmob` |
| `/linvsee <player>` | Просмотр инвентаря | `loutils.invsee` |
| `/lenchant <enchant> <level>` | Зачарование | `loutils.enchant` |
| `/lfly [player]` | Fly toggle | `loutils.fly` |
| `/lflyspeed <0-10> [player]` | Fly speed | `loutils.flyspeed` |
| `/ltpsbar [on\|off]` | TPS BossBar | `loutils.tpsbar` |
| `/worldlock <add\|remove\|list>` | Блокировка миров | `loutils.worldlock` |
| `/sit` | Сесть | `loutils.sit` |
| `/map <lock\|unlock>` | Блокировка карт | `loutils.map` |
| `/loutils reload` | Перезагрузка | `loutils.admin` |

### Права

Все права по умолчанию доступны только **OP** (кроме поз — они доступны всем).

**Обход блокировок:**
- `loutils.worldlock.bypass.*` — обход всех миров
- `loutils.worldlock.bypass.<world>` — обход конкретного мира
